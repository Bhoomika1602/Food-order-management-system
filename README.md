**# Food-order-management-system**
This is a simple Food Ordering Management System implemented in C programming language. The system allows users to view food items, place orders, and make payments. Administrators can add, delete, and update food items, view orders, and backup data.

**#Features**
User can view food items and place orders
User can make payments using cash or credit card
Administrator can add, delete, and update food items
Administrator can view orders and backup data
System displays a welcome message and a loading bar

**#Functions**
insertfirst: Inserts a new food item at the beginning of the list
insertmid: Inserts a new food item at a specified position in the list
insertend: Inserts a new food item at the end of the list
deletefood: Deletes a food item from the list
updatefood: Updates the quantity of a food item
countitem: Returns the number of food items in the list
foodlist: Displays the list of food items
order_view: Displays a single food item in a formatted way
ccolor: Sets the text color in the console
pwellcome, loadingbar, and backuploader: Display a welcome message, a loading bar, and a backup loader animation, respectively

**#How to run**
Compile the code using a C compiler (e.g., GCC)
Run the executable file

**#System Requirements**
C compiler (e.g., GCC)
Windows operating system (for console color functions)
